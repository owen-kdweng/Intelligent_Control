# Lesson7 Final Project

## 1. 期末作業主題
針對先前課程的差速移動機器人模擬器，設計、實作並驗證一個控制器。控制器可以是模糊控制、類神經網路控制、強化學習控制、適應控制、遺傳演算法調參控制，或其他與智慧型控制相關的方法。可自行設計不同難度之連續目標點，並且比較與PID方法的差異。

## 2. 學習目標
  1. 選擇、參考、創造一個受控系統模型。
  2. 比較自行實現之控制器與PID控制器的表現。
  3. 撰寫技術報告並進行成果展示。

## 3. 報告時間
第十六週

## 4. 評分規則

### 4.3 方法分數(40%)  
| 方法                              | 說明                                 | 分數       |
| -------------------------------- | ------------------------------------ | -----------|
| 傳統控制                          | 基於演算法並未結合其他方法             | 10以上     |
| 混合式控制(傳統方法)               | 例如 PID                             | 15以上     |
| 混合式控制                        | 例如 PID + RL、Fuzzy + NN             | 30以上     |
| 智慧型方法                        | 可挑選以下方法或其他先進方法            | 30以上     |

| 方法                              | 說明                                 |
| -------------------------------- | ------------------------------------ |
| 類神經網路控制 Neural Network Control | 使用 NN 近似控制律或模型                     |
| 強化學習控制 Reinforcement Learning  | 使用 Q-learning、DQN、DDPG、PPO 等方法     |
| 適應控制 Adaptive Control          | 控制參數可依系統狀態調整                       |
| 神經模糊控制 ANFIS                   | 結合模糊邏輯與神經網路                        |
| 遺傳演算法最佳化 GA-based Control      | 使用 GA、PSO 等方法調整 PID 或模糊控制器參數       |
| 智慧型 PID                        | 使用 AI 方法自動調整 PID 參數                |

### 4.4 方法比較分數(20%)
依照自行實現之方法與課堂方法(PID控制器)的比較完整度進行評分，建議使用性能指標具體展現。

### 4.5 報告與簡報表達(20%)
依照報告內容與狀況評分，不限但須至少包含方法理論解釋、實作相關工具與方法介紹、成果展示。報告時間需10至15分鐘內。

### 4.6 創意與難度(10%)
題目難度、方法創新度、整合程度。

### 4.7 其他(10%)
提交可執行程式、使用說明、模型檔與實驗紀錄。
